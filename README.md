# tbc_voxel_qsgs

本项目用于推进用户第一篇小论文：

**基于体素法的 EB-PVD 8YSZ 热障涂层跨尺度等效力学性能预测及其与 QSGS 方法对比。**

## Research Goal

基于论文 `热障涂层宏微观性能的跨尺度预测.pdf`，将原文中的 `Quartet Structure Generation Set (QSGS)` 微结构生成方法替换/扩展为 `voxel method`，并系统比较两种微结构重构方法对热障涂层跨尺度等效力学性能预测结果的影响。

核心研究问题：

1. 体素法能否生成满足目标孔隙率、方向性和尺寸控制要求的 8YSZ 热障涂层三维 RVE？
2. 在相同 RVE 尺寸、材料参数和 AHM 流程下，voxel method 与 QSGS 预测的等效力学性能有何差异？
3. 这些差异是否能从孔隙连通性、孔隙尺寸分布、各向异性和表面积/体积比等形貌指标解释？
4. 哪种方法在可控性、计算效率、随机稳定性和论文适用性上更优？

## Current Direction

当前正式方向由老师确定：

> 基于“热障涂层宏微观性能的跨尺度预测”这篇文章，将其中的 QSGS 方法换成体素法生成，并与 QSGS 进行对比，形成一篇小论文。

本项目不把“替换算法”当作唯一贡献，而是把论文主线提升为：

> 不同微结构重构方法对 EB-PVD 8YSZ 热障涂层跨尺度等效力学性能预测的影响。

## Baseline Paper Facts

从 `热障涂层宏微观性能的跨尺度预测.pdf` 已确认：

- 对象：EB-PVD 8YSZ 热障涂层。
- 原方法：`QSGS + AHM`。
- 原文 RVE 尺寸：`40 um x 40 um x 40 um`。
- 材料参数：`E = 48 GPa`，`nu = 0.12`，`rho = 3610 kg/m3`。
- 主要参数扫描：孔隙率、孔隙长细比、孔隙尺寸。
- 主要输出：等效杨氏模量、剪切模量、泊松比、各向异性变化。
- 验证方式：与 `K-A model` 杨氏模量结果对比。

## Expected Paper Story

建议论文主线：

1. EB-PVD 8YSZ 热障涂层孔隙结构复杂，微结构重构方法会影响跨尺度性能预测。
2. 原文 QSGS 能生成随机各向异性孔隙结构，但体素法在结构表达、图像重构和数值实现上具有潜力。
3. 本文在统一 AHM 框架下，比较 QSGS 与体素法生成的 RVE。
4. 比较内容包括形貌指标、等效力学性能、随机稳定性和计算效率。
5. 讨论两种方法在热障涂层结构优化中的适用范围。
