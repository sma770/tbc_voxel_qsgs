# 给老师师兄的问题清单_v1

用途：这份清单用于和老师/师兄沟通，目标不是显得“我全不会”，而是明确表达“我已经读过稿件，知道关键卡点，想拿材料做最小复现”。

## 1. 建议先发的简短消息

老师/师兄您好，我已经初步通读了两篇小论文，并整理了两条路线的主线和关键问题。现在我的理解是：

- TBC 裂纹方向主线更集中，适合先做最小复现；
- TSV 方向创新空间大，但需要电磁/热/力模型、RLGC 公式和 S 参数数据支撑。

为了下一步不是只停留在读稿和改文字，我想先拿到原始材料，复现其中一部分关键结果。请问方便发我一下相关模型、数据或代码吗？另外也想请老师/师兄帮我确认：如果我要在其中一篇基础上继续推进成我的第一篇小论文，哪些内容可以继承，哪些部分需要我重新做或补充？

## 2. 优先问老师的问题

这些问题适合问老师，因为它们涉及方向选择、创新边界和论文要求。

1. 老师更建议我优先从 TBC 裂纹方向还是 TSV 多场耦合方向继续做？
2. 如果作为我的第一篇小论文，老师希望我新增的工作量主要体现在哪里？
3. 师兄稿件中哪些内容我可以继承，哪些内容必须重新计算、验证或改写？
4. 对第一篇小论文，老师更看重“完整复现 + 扎实解释”，还是“新增模型/新增参数分析”？
5. TBC 方向里，老师更希望我补理论推导、补 RVE 仿真，还是补机器学习验证？
6. TSV 方向里，老师更希望我补多场仿真、补 S 参数分析，还是先纠正 RLGC 和图表逻辑？
7. 老师批注中哪些是必须修改，哪些只是建议优化？
8. 后续汇报时，老师希望我先汇报选题判断，还是先汇报一项最小复现结果？

## 3. 优先向 TBC 师兄索要的材料

这些问题适合直接问做 TBC 稿件的师兄。

### 3.1 RVE 与均匀化

1. 含裂纹 RVE 是用哪个软件建的？Abaqus、COMSOL、Python 脚本还是其他工具？
2. 裂纹几何是从真实显微图像提取，还是参数化生成？
3. 如果是真实图像提取，能否提供显微图、裂纹统计数据和提取方法？
4. 如果是参数化生成，能否提供裂纹密度、取向、纵横比等参数定义和生成脚本？
5. 周期性边界条件 PBC/MPC 是怎么施加的？有没有脚本或模型文件？
6. 六个独立单位预定义应变工况是如何设置的？
7. 等效刚度矩阵最终是用哪个公式或程序算出来的？
8. 老师批注提到“到底用哪个计算”，师兄当时最终采用的是均匀化积分公式，还是虚功/节点力公式？

### 3.2 数据与图表

1. 网格无关性验证表的原始数据能否提供？
2. AHM 方法适用性分析中，Budiansky and O'Connell、Kato/Nishioka 的对比曲线数据来自哪里？
3. 裂纹密度、裂纹角、纵横比影响图的原始数据能否提供？
4. 这些图是用 Origin、Python、MATLAB 还是其他工具画的？
5. 老师批注中提到的方格图样式、图例和重叠曲线，是否有已修改版本？

### 3.3 XGBoost

1. XGBoost 的原始数据集能否提供？每一列分别是什么含义？
2. 12 个等效力学性能参数分别是哪 12 个？
3. 训练集和测试集如何划分？随机种子是多少？
4. 是否有验证集或交叉验证结果？
5. 超参数随机搜索的代码能否提供？
6. 特征重要性是用哪种方法算的？gain、weight、cover、SHAP，还是其他？
7. 是否尝试过 Random Forest、SVR、MLP 等对比模型？

## 4. 优先向 TSV 师兄索要的材料

这些问题适合直接问做 TSV 稿件的师兄。

### 4.1 模型与软件

4. 材料参数的温度依赖关系来自哪里？

5. 趋肤深度和交流电阻计算脚本能否提供？
6. 图 3 的等效电路和阻抗网络是否有原始绘图或推导说明？

## 5. 面谈时的推荐顺序

如果只有 10 分钟，建议这样问：

1. 老师，我读下来觉得 TBC 更适合作为第一篇小论文主线，您看这个判断对吗？
2. 如果做 TBC，我应该优先补“RVE 均匀化复现”还是“XGBoost 验证改进”？
3. 师兄能否把 TBC 的 RVE 模型、均匀化脚本、数据集和训练脚本发我？
4. 师兄稿中等效刚度到底用哪个公式计算？老师批注这块我想先解决。
5. 我后续新增工作量是否可以定为：复现关键图 + 补充泛化验证 + 重画并解释特征重要性？

如果老师更倾向 TSV，再问：

1. TSV 的低频热力模型和高频 S 参数模型能否拿到？
2. `S11/S21` 目前图题和正文是否需要整体重新核对？
3. 如果我做 TSV，是否只做一个小闭环，比如温度对 RLGC 和 S 参数影响，而不是全链路都做？

## 6. 你自己的沟通底线

和老师/师兄沟通时，建议你坚持这三点：

1. **不要只接“改文字”的任务。**
   你的第一篇小论文需要自己的新增工作量，至少要有复现、补分析、补验证或重画数据图。

2. **先要原始材料。**
   没有模型、数据、脚本，只读 Word 很难判断能不能做。

3. **先做最小闭环。**
   TBC 就先复现一个裂纹参数影响图或 XGBoost 结果；TSV 就先复现一个温度/RLGC/S 参数影响链。
