# 研究方向备忘

## 主线方向

**不同微结构重构方法对 EB-PVD 8YSZ 热障涂层跨尺度等效力学性能预测的影响。**

核心对比：

- `QSGS`：原文采用的四参数随机生长法。
- `voxel method`：本项目拟引入的体素法。
- 统一后续框架：`RVE -> AHM -> effective mechanical properties`。

主线不要写成“把 QSGS 换成体素法”，而要写成：

> 在相同 RVE 尺寸、材料参数和 AHM 求解流程下，比较 QSGS 与体素法生成的微结构在形貌特征、等效力学性能、计算效率和随机稳定性上的差异。

## 优先拓展方向

这些方向性价比高，适合补强小论文质量：

1. **随机样本统计**
   同一孔隙率下生成多个 RVE，输出均值、标准差和误差条。

2. **形貌指标对比**
   比较孔隙率、连通性、孔隙尺寸分布、各向异性和表面积/体积比。

3. **K-A 模型验证**
   用原文表格或 K-A 经验模型做 sanity check，证明结果不离谱。

4. **体素分辨率敏感性**
   比较不同 voxel resolution 对形貌指标和等效性能的影响。

5. **计算效率对比**
   比较 QSGS 与体素法的生成时间、数据规模和后处理复杂度。

## 条件满足后再做

这些方向有价值，但需要额外材料或时间：

1. **基于真实显微图像的 voxel reconstruction**
   如果老师/师兄能提供 EB-PVD 8YSZ 显微图，可升级为 image-based voxel reconstruction。

2. **RVE 尺寸收敛对比**
   比较 `20/30/40/50 um` 下两种方法的收敛性。

3. **热导率或热-力耦合扩展**
   原文主要做力学性能；若时间充足，可扩展到热导率或热应力。

4. **裂纹/孔隙耦合缺陷**
   后续可引入裂纹，但第一篇不建议一开始加入，容易失控。

## 暂不优先做

以下方向容易让第一篇小论文过大：

- 深度生成模型或扩散模型生成微结构。
- 完整服役过程中的孔隙演化、裂纹扩展和高温蠕变。
- 涂层-基体界面失效全过程。
- 大规模机器学习预测，除非已有足够数据。
