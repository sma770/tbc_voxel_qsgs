## 1. 原文研究主线

原文面向 EB-PVD 制备的 8YSZ 热障涂层，核心问题是：真实涂层内部存在随机、不规则、各向异性的孔隙/裂纹等微结构缺陷，传统理想化孔隙模型难以准确反映微结构对宏观力学性能的影响。

原文方法主线可以拆成：

1. 使用四参数随机生长法 QSGS 构建具有空间随机性和各向异性特征的三维孔隙微结构。
2. 将生成的微结构作为周期性 RVE。
3. 使用渐近均匀化方法 AHM 计算等效刚度张量。
4. 从等效刚度张量提取等效杨氏模量、剪切模量、泊松比等性能。
5. 扫描孔隙率、孔隙长细比、孔隙尺寸等微结构参数，分析其对等效力学性能和各向异性的影响。
6. 用 K-A 模型与 AHM 得到的杨氏模量做验证，证明数值结果在合理范围内。

可复用为本项目的小论文主线：

> 在相同 RVE 尺寸、材料参数、边界条件和 AHM 求解流程下，比较 QSGS 与体素法生成的 EB-PVD 8YSZ 热障涂层微结构在形貌指标、等效力学性能、随机稳定性和计算效率上的差异。

## 2. QSGS 参数表

### 2.1 四参数定义

| 参数 | 原文名称 | 含义 | 复现关注点 |
| --- | --- | --- | --- |
| `Pc` | Core Distribution Probability，核心分布概率 | 控制初始生长核密度；`Pc` 越大，生长核越多，孔隙尺寸通常越小、分布更均匀 | 需要确认原文实现中 `Pc` 是作用于固相、生长相还是孔隙相 |
| `Di` | Directional Growth Probability，方向生长概率 | 控制 26 个邻近方向上的扩展概率；所有 `Di` 相等时近似各向同性，不相等时产生方向性 | 重点复现柱状晶方向 `D2 = D4` 的设置 |
| `Pn` | Volume fraction，体积分数 | 控制目标生长相体积分数；原文中孔隙率和固相体积分数互补 | 需要统一输出实际孔隙率和目标孔隙率误差 |
| `Pirm` | Phase Interaction growth Probability，相间生长概率 | 多相材料中不同相之间的相互作用概率 | 当前论文主要是固相/孔隙二值结构，若原文没有给数值，可先标记为待确认 |

## 3. AHM 流程拆解

原文 AHM 的角色是把周期性 RVE 的微观非均质结构转化为宏观等效弹性性能。复现时建议按工程实现流程拆解，而不是一开始追求公式完全展开。

### 3.1 输入

- 三维 RVE 几何：固相为 8YSZ，孔隙为空相/近似极低刚度相。
- 固相材料参数：`E = 48 GPa`，`ν = 0.12`。
- RVE 尺寸：`40 μm × 40 μm × 40 μm`。
- 周期性边界条件 PBC。
- 六个独立单位宏观应变场。

### 3.2 有限元/均匀化计算步骤

1. 将 QSGS 或体素法生成的二值结构转换为可计算模型。
2. 给固相赋予 8YSZ 弹性参数；孔隙可设为空洞、极低刚度材料，或在有限元中删除孔隙单元，具体取决于求解器实现。
3. 对 RVE 施加周期性边界条件，使相对位移满足宏观平均应变与周期扰动位移关系。
4. 依次施加六个独立单位初始应变场：
   - `ε11 = 1`
   - `ε22 = 1`
   - `ε33 = 1`
   - `γ23 / ε23` 剪切场
   - `γ13 / ε13` 剪切场
   - `γ12 / ε12` 剪切场
5. 每个工况求解特征位移/扰动位移场。
6. 对 RVE 内应力或能量进行体积平均，得到等效刚度矩阵 `C*`。
7. 由 `C*` 或柔度矩阵 `S* = inv(C*)` 提取：
   - `E1, E2, E3`
   - `G12, G23, G13`
   - `ν12, ν21, ν23, ν32, ν13, ν31`
8. 对结果做归一化与对比，例如 `E* = E_eff / E_solid`、`G* = G_eff / G_solid`。

### 3.3 第一轮 AHM 复现策略

优先级从低风险到高完整度：

1. 先复现原文 K-A/AHM 表中的孔隙率-杨氏模量趋势，至少拿到 10% 孔隙率附近结果。
2. 若没有原始 AHM 脚本，先实现一个最小 PBC 均匀化模型或使用已有有限元平台模板。
3. 若 PBC 实现短期卡住，先完成微结构生成与形貌指标，并用 K-A 模型做临时 sanity check。
4. 记录每次求解器版本、网格/体素分辨率、孔隙处理方式和边界条件，因为这些会显著影响复现偏差。

## 6. 体素法替代位置

体素法应替代/扩展的是“微结构生成”模块，而不是替代整个跨尺度流程。

建议项目方法框架：

```text
参数输入
  ├─ QSGS generator
  │    └─ binary voxel RVE
  └─ voxel-method generator
       └─ binary voxel RVE

统一后处理
  ├─ 形貌指标计算
  ├─ 可视化
  ├─ AHM/PBC 等效性能计算
  └─ 统计分析与误差条
```

体素法的位置：

- 输入层：使用与 QSGS 一致的目标孔隙率、RVE 尺寸、分辨率、随机种子策略。
- 生成层：体素法直接生成三维 binary RVE，或基于图像/规则/随机场生成后体素化。
- 输出层：必须转换成与 QSGS 完全相同的数据结构，例如 `0 = pore`、`1 = solid`。
- AHM 层：不允许为体素法单独改变材料参数、边界条件或等效性能提取方式。

需要尽快向老师确认：“体素法”具体指哪一种：

1. 随机体素生成：直接在 3D 网格中控制孔隙率和方向性。
2. 图像体素重构：基于真实 SEM/CT/显微图像重构。
3. 几何体素化：先生成柱状孔/裂纹几何，再离散到体素网格。
4. 某篇文献中的特定 voxel method。

## 7. 公平对比条件

QSGS 与体素法对比时必须锁定以下条件：

| 条件 | 固定方式 | 目的 |
| --- | --- | --- |
| RVE 尺寸 | 同为 `40 μm × 40 μm × 40 μm` | 排除尺寸效应 |
| 体素分辨率 | 同一网格数或同一体素边长 | 排除离散误差 |
| 目标孔隙率 | 同一目标值，并报告实际值 | 排除体积分数差异 |
| 材料参数 | 固相 `E = 48 GPa`，`ν = 0.12` | 排除材料差异 |
| 孔隙处理 | 同为空洞或同为极低刚度相 | 排除求解设置差异 |
| 边界条件 | 同一 PBC/AHM 设置 | 排除边界条件差异 |
| 求解器参数 | 同一单元类型、网格策略、收敛准则 | 排除数值误差 |
| 随机样本数 | 每组同样数量随机种子 | 支持误差条对比 |
| 输出指标 | 同一形貌指标和同一等效性能提取公式 | 保证可比性 |
| 计算效率 | 记录生成时间、求解时间、内存/文件大小 | 支持方法适用性讨论 |

公平对比的底线：体素法可以在微结构形貌上不同，但进入 AHM 之前的数据格式、材料参数和求解流程必须一致。
