## 4. AHM 流程整理

### 1. RVE 建立

1. 使用 `QSGS` 生成三维随机多孔 RVE。

2. RVE 采用 binary 表示：

   - `1 = solid 8YSZ`
   - `0 = pore`

3. 基准物理尺寸为 `40 μm × 40 μm × 40 μm`。

4. NotebookLM 记录的尺寸收敛依据为：

   `||C^{N+1} - C^N|| / ||C^N|| < Tol`

   其中 `Tol = 0.02`，并通过 `5–50 μm` 尺寸模型的均匀化结果确定 `40 μm` RVE。该公式、容差和尺寸扫描范围均需回原文 PDF 核验。

### 2. 材料参数

RVE 中固相 8YSZ 的材料参数暂按原文基准值记录：

| 参数     | 符号  | 数值   | 单位    | 当前状态 |
| -------- | ----- | ------ | ------- | -------- |
| 杨氏模量 | `E`   | `48`   | `GPa`   | 已记录   |
| 泊松比   | `nu`  | `0.12` | `-`     | 已记录   |
| 密度     | `rho` | `3610` | `kg/m³` | 已记录   |

说明：

1. 孔隙相的力学处理方式待核验，例如是否为空洞、极低刚度相、删除单元或其他数值处理方式。
2. 第一轮复现需要统一单位体系，避免 `μm`、`GPa`、`Pa` 混用导致数值错误。

### 3. 边界条件 / 单位应变加载

1. 原文 RVE 边界使用 `Periodic Boundary Conditions (PBCs)`。

2. NotebookLM 记录的 PBC 位移约束形式为：

   `u_dst = u_src + εbar(r_dst - r_src)`

   其中：

   - `u_dst`：目标面节点位移；
   - `u_src`：源面节点位移；
   - `εbar`：宏观平均应变；
   - `r_dst - r_src`：周期对应点之间的位置差。

3. 等效刚度计算中需施加六个独立单位应变工况：

   - 3 个正应变工况：`ε11`、`ε22`、`ε33`
   - 3 个剪应变工况：`ε12`、`ε23`、`ε13`

待核验内容：

1. 原文中源面/目标面如何配对。
2. 角点、棱边、面内节点的 PBC 约束如何处理。
3. 刚体位移如何消除。
4. 剪应变采用工程剪应变还是张量剪应变。
5. 有限元软件或脚本中 PBC 的具体实现方式。

### 4. 局部场求解

根据 NotebookLM 输出，原文基于 `AHM` 进行局部场求解，核心思想包括：

1. 引入宏观尺度变量 `x` 和微观尺度变量 `y = x / ξ`。

2. 对位移场进行多尺度渐近展开：

   `u_i^ξ(x) = u_i^0(x) + ξ u_i^1(x,y) + ξ^2 u_i^2(x,y) + ...`

3. 引入位移特征函数 `χ_k^{mn}` 描述微观扰动。

4. 将特征函数问题离散为有限元线性方程：

   `Kχ^{kl} = f^{kl}`

待核验内容：

1. 原文特征函数控制方程的完整推导和符号定义。
2. `χ_k^{mn}` 的周期性条件。
3. 孔隙相在刚度矩阵中的处理方式。
4. 原文使用的单元类型、网格尺寸和求解器设置。

### 5. 等效刚度矩阵提取

NotebookLM 记录的流程为：

1. 对 RVE 依次施加六个独立单位应变工况。
2. 每个工况下求解局部位移场。
3. 根据局部位移场计算局部应变和局部应力。
4. 对局部应力/刚度响应进行体积平均。
5. 组装得到完整等效宏观刚度张量或等效刚度矩阵。

等效弹性张量公式的 NotebookLM 记录形式为：

```
C_ijmn^H = 1/|Y| ∫_Y (C_ijmn + C_ijkl ∂χ_k^{mn}/∂y_l) dY
```

待核验内容：

1. 原文公式中正负号、指标顺序和求导变量是否与 NotebookLM 输出一致。
2. 刚度矩阵从四阶张量转为 Voigt 矩阵时的顺序。
3. 剪切项是否涉及工程剪应变系数。
4. 体积积分是否只在固相区域进行，还是在整个 RVE 上进行并将孔隙按零刚度处理。

## 5. K-A model 验证方式

NotebookLM 记录的验证方式如下：

1. 控制孔隙率在 `2%–20%` 范围内变化。
2. 使用 `AHM` 计算不同孔隙率下的等效杨氏模量。
3. 使用 `K-A model` 计算对应孔隙率下的等效杨氏模量。
4. 比较两者误差。
5. NotebookLM 记录误差范围为 `1.2%–9.4%`，并提到 `16%` 孔隙率时误差约 `9.4%`。

当前状态：

1. `K-A model` 的具体公式：待核验。
2. `K-A model` 中各参数含义：待核验。
3. 原文对比表格或图中具体数据：待核验。
4. `1.2%–9.4%` 的误差范围：待回原文表格核验。
5. 是否只验证某一方向杨氏模量，还是验证多个方向等效杨氏模量：待核验。

M1 阶段只记录验证路线，不直接进行 K-A model 计算。

## 7. voxel method 替代位置

`voxel method` 将替代或扩展原文流程中的 **RVE 微结构生成环节**。

原文流程可概括为：

```
QSGS 参数 → 三维 porous RVE → AHM/PBC 均匀化 → 等效刚度矩阵 → E/G/nu → K-A model 验证
```

本项目后续拟改为：

```
QSGS RVE 与 voxel RVE → 相同 AHM/PBC 均匀化 → 相同 E/G/nu 输出 → 形貌指标与等效性能对比
```

替代位置说明：

1. 保持 RVE 物理尺寸一致。
2. 保持目标孔隙率一致。
3. 保持材料参数一致。
4. 保持 AHM/PBC 流程一致。
5. 保持后处理指标一致。
6. 只改变或扩展微结构生成方法。
7. 后续比较 `QSGS` 与 `voxel method` 生成结构在孔隙率、孔隙连通性、孔隙尺寸、方向性、表面积/体积比、等效力学性能和稳定性上的差异。

重要待确认：

老师所说的 `voxel method` 具体含义尚未确认，可能包括：

1. 随机体素生成；
2. 基于显微图像的 voxel reconstruction；
3. 将已有几何结构离散为 voxel representation。

在老师未确认前，不能把任何一种解释写成最终事实。
