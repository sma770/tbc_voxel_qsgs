# QSGS 实现假设 v1

## 1. 文件目的

本文档用于记录 `tbc_voxel_qsgs` 项目中第一轮最小 QSGS 复现的实现假设。

当前目标不是一次性完成论文级完整复现，而是先建立一个透明、可测试、可复现的 QSGS baseline，为后续与 Anisotropic GRF 随机体素法进行公平对比做准备。

本文件中的实现假设后续可根据原文进一步核验结果、老师意见或师兄资料进行修订。

---

## 2. 相编码规则

本项目统一采用二值 RVE 表示：

- `1 = solid 8YSZ phase`
- `0 = pore phase`

在 QSGS 生成过程中，固相 8YSZ 作为生长相，孔隙相作为非生长相。

也就是说，QSGS 生长过程本质上是：

```text
从少量 8YSZ 固相生长核开始
→ 按方向概率向周围孔隙体素生长
→ 直到固相体积分数达到目标值
→ 剩余未生长区域为孔隙
