# 体素法方案 v1

生成日期：2026-06-09

当前阶段：等待师兄提供 QSGS/AHM 代码期间，先完成 `voxel method` 的方法定义、文献依据和最小实现方案。

## 0. 方案结论

本项目中，`voxel method` 不建议定义为“在 3D 网格中独立随机填充孔隙”。那种方法只能控制孔隙率，难以控制孔隙尺寸、连通性和 EB-PVD 涂层的柱状方向性，作为主方法支撑论文偏弱。

建议将本文体素法定义为：

> 各向异性相关随机场体素生成法，英文暂称 `anisotropic correlated voxel method`。

核心思想：

```text
3D random field
-> anisotropic filtering / smoothing
-> threshold by target porosity
-> optional morphology adjustment
-> binary voxel RVE
```

这样做的好处是：

- 保留 voxel method 的直接、简单、易接 AHM 的优势。
- 不只控制孔隙率，还能控制孔隙相关长度、方向性和连通趋势。
- 与 QSGS 有清楚区别：QSGS 是 stochastic growth，本文 voxel method 是 correlated field + threshold。
- 能自然组织公平对比：同一 RVE、同一孔隙率、同一材料参数、同一 AHM/PBC 流程。

## 1. 方案是什么

### 1.1 方法定位

本文拟比较两类微结构重构方法：

| 方法 | 类型 | 论文定位 |
| --- | --- | --- |
| QSGS | stochastic growth reconstruction | 原文基准方法 |
| simple random voxel | independent voxel sampling | 低级对照，证明只控制孔隙率不够 |
| anisotropic correlated voxel method | correlated random-field voxel reconstruction | 本文主对比方法 |

第一篇小论文的推荐主线：

> 在相同 RVE 尺寸、孔隙率、材料参数、边界条件和 AHM 后处理流程下，比较 QSGS 与各向异性相关体素法生成的微结构在形貌指标和等效力学性能上的差异。

### 1.2 体素法输出

所有微结构生成方法都必须输出统一格式：

```text
RVE array: shape = (Nx, Ny, Nz)
0 = pore
1 = solid

metadata:
  method
  seed
  target_porosity
  actual_porosity
  rve_size_um
  voxel_size_um
  grid_shape
  parameters
  created_at
```

推荐保存为 `.npz`：

```text
data/processed/rve/<method>/<case_id>.npz
```

## 2. 为什么这么做

### 2.1 只做 simple random voxel 不够

simple random voxel 的做法是：每个 voxel 独立按概率变成 pore 或 solid。它适合作为 baseline，但不适合作为主方法。

主要问题：

- 孔隙容易碎片化，不像 EB-PVD 8YSZ 中具有方向性的柱状孔隙/间隙。
- 只能精确控制孔隙率，难以控制孔隙尺寸和相关长度。
- 方向性需要额外机制，否则很难解释各向异性等效性能。
- 与 QSGS 相比太弱，容易被老师质疑“只是随机撒点”。

### 2.2 相关随机场体素法更适合

各向异性相关随机场体素法先生成一个连续随机场，再通过各向异性平滑引入方向相关性，最后通过阈值控制孔隙率。

它能控制：

- `target_porosity`：目标孔隙率。
- `correlation_length_x/y/z`：三方向相关长度。
- `anisotropy_ratio`：方向性强弱，例如柱状方向相关长度更大。
- `seed`：随机样本。
- `morphology_adjustment`：可选的开闭运算、连通性修正。

这比 simple random voxel 更接近“微结构重构方法”，而不是“随机填充算法”。

### 2.3 与 QSGS 的互补关系

QSGS 通过核心分布概率和方向生长概率形成结构，强调“生长过程”；体素法通过相关随机场和阈值形成结构，强调“体素场表达和统计控制”。

| 对比点 | QSGS | anisotropic correlated voxel method |
| --- | --- | --- |
| 生成机制 | 随机种子 + 方向生长 | 随机场 + 各向异性滤波 + 阈值 |
| 方向性控制 | 方向生长概率 `Di` | 三方向相关长度或滤波核 |
| 孔隙率控制 | 目标体积分数 `Pn` | 阈值分位数 |
| 孔隙尺寸控制 | `Pc` 和生长规则间接控制 | 相关长度直接控制 |
| 与 AHM 连接 | 输出 binary RVE 后连接 | 天然输出 voxel RVE |
| 论文价值 | 原文基准 | 新增可控对比方法 |

## 3. 别人怎么做

### 3.1 QSGS：随机生长型重构

QSGS 的文献基础来自随机多孔介质结构生成。Wang 等在 2007 年的工作中使用 QSGS 构建微尺度随机多孔介质，并预测有效热导率。后续文献也常把 QSGS 描述为基于 stochastic cluster growth 的随机内部形貌生成方法，其四参数通常包括核心分布概率、方向生长概率、相体积分数和相互作用生长概率。

本项目使用方式：

- 把 QSGS 作为原文基准。
- 不把 QSGS 本身写成本文创新。
- 复现后只将其作为公平对比的一支。

代表文献：

- Moran Wang, Jinku Wang, Ning Pan, Shiyi Chen, 2007, *Physical Review E*, `Mesoscopic predictions of the effective thermal conductivity for microscale random porous media`, DOI: `10.1103/PhysRevE.75.036702`。
- QSGS 方法综述式应用可参考 2025 年 porous media/pore-throat network 工作中对 QSGS 四参数和随机生长过程的描述。

### 3.2 digital image / voxel-based homogenization：图像或体素到有限元

Terada、Kikuchi 等较早系统提出了 digital image-based modeling 与 homogenization 的结合：将图像中的 pixel/voxel 识别为固定尺寸有限元，堆叠 2D 图像得到 3D voxel unit cell，再进行均匀化分析。这说明“体素表示 + homogenization”本身是有文献基础的。

本项目使用方式：

- 说明 voxel method 的优势：二值 RVE 可直接进入规则网格或体素有限元。
- 如果后续有 SEM/CT 图像，可扩展为 image-based voxel reconstruction。
- 当前没有真实图像时，先做 synthetic voxel RVE。

代表文献：

- Kenjiro Terada, Noboru Kikuchi, 1998, `Microstructural Modeling Technique for Homogenization Analysis Using Digital Images`, DOI: `10.1299/kikaia.64.170`。
- Terada, Miura, Kikuchi, 1997, `Digital image-based modeling applied to the homogenization analysis of composite materials`, *Computational Mechanics*, DOI: `10.1007/s004660050255`。

### 3.3 Gaussian random field：相关随机场生成孔隙结构

Hyman 和 Winter 的工作给了本项目体素法很直接的理论依据：先通过卷积核生成 correlated Gaussian field，再用阈值把点分成 void/solid。卷积核的内在长度尺度决定相关结构，阈值决定孔隙体积分数。这一思路可以自然扩展到各向异性：让不同方向的滤波核尺度不同。

本项目使用方式：

- 将 `correlation_length_x/y/z` 作为体素法核心参数。
- 通过阈值精确控制目标孔隙率。
- 用方向相关长度模拟 EB-PVD 柱状方向的孔隙连通或拉长趋势。

代表文献：

- Jeffrey D. Hyman, C. Larrabee Winter, 2014, `Stochastic generation of explicit pore structures by thresholding Gaussian random fields`, *Journal of Computational Physics*, DOI: `10.1016/j.jcp.2014.07.046`。

### 3.4 anisotropic microstructure generation：统计量与各向异性

近年材料信息学和 microstructure-sensitive modeling 中，常用 1-point/2-point statistics、Gaussian random fields 或类似统计生成模型来生成具有指定统计特征的微结构。相关文献强调 2-point statistics 能表达微结构的方向性和空间相关性。

本项目不建议引入深度生成模型或复杂 inverse design，因为第一篇小论文的工作量会失控；但可以借用这类文献的思想：用低阶统计量和相关长度控制微结构。

代表文献：

- Andreas E. Robertson, Surya R. Kalidindi, 2022, `Efficient generation of anisotropic N-field microstructures from 2-point statistics using multi-output Gaussian random fields`, *Acta Materialia*, DOI: `10.1016/j.actamat.2022.117927`。

### 3.5 EB-PVD 8YSZ 背景：为什么要控制方向性

EB-PVD 热障涂层具有典型柱状微结构，微结构、孔隙和柱状形貌会影响热学和力学性能。因此，本文 voxel method 不能只匹配孔隙率，还应能表达方向性、孔隙尺度和连通性。

代表文献：

- `Effect of substrate rotation on the microstructure of 8YSZ thermal barrier coatings by EB-PVD`, *Materials Today: Proceedings*, DOI: `10.1016/j.matpr.2019.12.276`。
- `Microstructure Dependence of Effective Thermal Conductivity of EB-PVD TBCs`, 2021，说明 EB-PVD coating 的 columnar microstructure 与性能关联。

## 4. 怎样做

### 4.1 第一轮方法矩阵

建议第一轮只做 3 种方法：

| 方法编号 | 方法 | 目的 |
| --- | --- | --- |
| M0 | QSGS | 原文基准，等待师兄代码或复现 |
| M1 | simple random voxel | 低级对照，说明只控制孔隙率不够 |
| M2 | anisotropic correlated voxel | 本文主方法 |

### 4.2 anisotropic correlated voxel method 算法

输入：

```text
grid_shape = (Nx, Ny, Nz)
rve_size_um = (40, 40, 40)
target_porosity = p
seed = s
correlation_length = (lx, ly, lz)
phase_convention = 0 pore, 1 solid
```

步骤：

1. 生成标准正态随机场 `noise(x, y, z)`。
2. 使用各向异性 Gaussian filter：
   - `sigma_x = lx / voxel_size`
   - `sigma_y = ly / voxel_size`
   - `sigma_z = lz / voxel_size`
3. 得到平滑随机场 `field(x, y, z)`。
4. 按目标孔隙率取分位数阈值：
   - 如果孔隙取低值区，则 `threshold = quantile(field, target_porosity)`。
   - `pore = field <= threshold`。
5. 可选形貌修正：
   - 去除极小孤立孔隙。
   - 形态学开/闭运算。
   - 周期边界拼接检查。
6. 输出二值 RVE：
   - `0 = pore`
   - `1 = solid`
7. 记录 metadata。

### 4.3 参数初值建议

先不要追求物理完全准确，第一轮用参数扫描看趋势：

| 参数 | 建议初值 | 说明 |
| --- | --- | --- |
| `rve_size_um` | `(40, 40, 40)` | 与原文一致 |
| `grid_shape` | `(80, 80, 80)` 或 `(100, 100, 100)` | 先兼顾速度和可视化 |
| `target_porosity` | `0.06, 0.10, 0.14` | 第一轮三个孔隙率点 |
| `seed_count` | `3`，后续扩到 `5` | 先做小样本 |
| `correlation_length_x` | `1-3 um` | 横向孔隙尺度 |
| `correlation_length_y` | `3-8 um` | 柱状方向可设更大；具体方向需与坐标定义统一 |
| `correlation_length_z` | `1-3 um` | 横向或厚度方向，需与原文坐标统一 |
| `anisotropy_ratio` | `1, 2, 4` | 做方向性敏感性 |

注意：坐标方向必须和原文 `D2 = D4` 所代表的柱状方向统一。当前这是待师兄代码/原文图确认的问题。

### 4.4 第一轮只实现到形貌指标

在师兄 AHM 代码到手前，先做：

```text
voxel generator
-> binary RVE
-> morphology metrics
-> visualization
```

建议指标：

- `actual_porosity`
- `pore_connected_components`
- `largest_pore_component_fraction`
- `solid_connected_components`
- `surface_area_to_volume`
- `directional_two_point_correlation`
- `estimated_correlation_length_x/y/z`
- `anisotropy_index`

这些指标能支撑后续解释：

> 在相同孔隙率下，不同重构方法改变了孔隙连通性、方向相关长度和表面积/体积比，从而导致 AHM 预测的等效刚度和各向异性不同。

## 5. 与 QSGS 的公平对比设计

### 5.1 固定条件

| 条件 | 固定方式 |
| --- | --- |
| RVE 尺寸 | `40 um x 40 um x 40 um` |
| 材料参数 | `E = 48 GPa`, `nu = 0.12` |
| 孔隙率 | `6%, 10%, 14%` 起步 |
| 输出格式 | 统一 binary RVE |
| AHM 流程 | 同一 PBC、同一后处理 |
| 样本数 | 每个方法/孔隙率相同 seed 数 |
| 统计方式 | 均值、标准差、误差条 |

### 5.2 可比较参数映射

QSGS 与体素法参数不是一一对应，但可以建立“功能对应”：

| 功能 | QSGS 参数 | voxel method 参数 |
| --- | --- | --- |
| 控制孔隙率 | `Pn` 或目标体积分数 | `target_porosity` + threshold |
| 控制孔隙尺寸 | `Pc` | `correlation_length` |
| 控制方向性 | `Di` | `anisotropy_ratio`, `correlation_length_y / correlation_length_x` |
| 控制随机性 | random seed | random seed |
| 控制连通性 | 生长规则、`Di`、`Pc` | correlation length + morphology adjustment |

论文里不要说二者参数“完全等价”，而要说它们在功能上对应。

## 7. 风险与边界

### 7.1 风险

- 如果体素法只控制孔隙率，创新性不足。
- 如果不做方向性，难以对应 EB-PVD 柱状结构。
- 如果不做多随机种子，结果会被随机波动影响。
- 如果 AHM 与 QSGS/voxel 输入格式不统一，公平性会被质疑。
- 如果引入真实图像、深度生成模型或 inverse design，第一篇工作量会失控。

### 7.2 边界

第一篇小论文暂不做：

- deep learning microstructure generation。
- diffusion model。
- 完整 image-based reconstruction，除非老师/师兄提供现成 SEM/CT 数据。
- 热-力耦合或服役演化。
- 裂纹扩展和高温蠕变。
